# 🧦 VueShopLearn - Tienda de Calcetines
Proyecto desarrollado con **Vue 3** y **Vite**.

## 🚀 Características
- Arquitectura de componentes Single File Components (.vue).
- Uso de **Axios** para consumo de API.
- Estilos personalizados con animaciones CSS.
- Simulación de backend con **JSON Server**.

## 🛠️ Instalación
1. `npm install`
2. `npm run dev`
